Utilizando tecnologias de desarrollo web

Enfoque mi atencion en el aprendizaje completo de HTML de forma didacta por internet mediante videos de Youtube, cursos online y practica, mucha practica como debe ser con el objetivo de aprender a escribir codigo de forma natural. Una vez realizadas muchas practicas y haber practicado"maquetar" con HTML, como es claro, mi siguiente objetivo fue CSS.

CSS fue el siguiente desafio a seguir y aunque a este lenguaje le sigo indagando fue y es una muy linda experiencia las oportunidades son muchisimas a la hora de maquetear con CSS por lo que continuo aprendiendo y observando como se trabaja con este lenguaje y a utilizar las diversas herramientas que se aplican a este como Librerias, Frameworks, SASS,SCSS.

Actualmente me encuentro desarrollando ambas junto a otras tecnologias
